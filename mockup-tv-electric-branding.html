Design Notes — v3

  • Lightning bolts are jagged SVG polylines with randomised zigzag segments — Tesla coil / electrical discharge feel
  • Bolts regenerate every 4 seconds so the path changes, creating a living, crackling effect
  • Each bolt has 3 layers: ambient glow (wide, blurred), outer glow (medium blur), and a bright core line
  • Short branch forks randomly flicker off the main bolt — like real electrical discharge branching
  • Spark nodes flash at random junction points along the bolt
  • Colour palette: purple (#8B5CF6) → blue (#60a5fa) → white — pulses between them
  • The Scream Meter now uses 1-6 Scream Skull icons instead of a /10 numeric bar
  • Skulls are neon red (#DC2626) with animated glowing drop-shadow — matches concept art
  • Unfilled skulls sit at 15% opacity to show the max scale
  • Each card now includes the attraction logo (top-left) and a brief description of the maze
  • Cards ranked #1 to #6, most scary to least scary — mazes only (no shows), reading left-to-right, top-to-bottom
  • Skull ratings: Night Terrors (6), Drowned (5), Westlake (4), Bunker (3), Strings (2), Signal Loss (1)
  • All animation is CSS + lightweight SVG — no canvas, no heavy JS, safe for TV hardware
